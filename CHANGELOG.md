## [2.5.0] - 2026-07-06

### ✨ أُضيف (Added)
- **مكتبة موسيقية مدمجة** — 18 أغنية (عربي كلاسيكي + حديث + إنجليزي)
- **محرك بحث ذكي للأغاني** — Weighted Scoring System بأوزان مخصصة
- **مشغل موسيقى متكامل** — Play, Pause, Seek, Progress Bar, Timer, Download
- **اقتراح بالمزاج** — AI يقترح أغاني بناءً على حالة المستخدم المزاجية
- **دعم URL Encoding** — أسماء ملفات عربية تعمل بدون مشاكل
- **توثيق احترافي** — README, CONTRIBUTING, CHANGELOG, SECURITY, CODE_OF_CONDUCT

### 🔧 أُصلح (Fixed)
- إصلاح كسر الكود بسبب Unicode Decoding (`%60` → backtick)
- إصلاح مشكلة تشغيل الأغاني ذات الأسماء العربية على Cloudflare
- تصحيح أخطاء إملائية في System Prompt العربي

### 🔄 تغيّر (Changed)
- تحسين System Prompt — أكثر دقة واختصاراً
- تحسين AI Logic — أغنية واحدة فقط في كل رد
- ترقية بنية المشروع مع ملفات توثيق كاملة

## [2.0.0] - 2026-07-05

### ✨ أُضيف (Added)
- **نظام محادثات متعددة** — إنشاء، تسمية، حذف، والتنقل بين المحادثات
- **لوحة تحكم أدمن** — تحليلات شاملة مع Firebase Firestore
- **رسوم بيانية** — نشاط يومي، توزيع اللغات والأجهزة
- **وضع فاتح/داكن** — انتقال سلس بين الوضعين
- **Markdown Parser مدمج** — عرض أكواد، قوائم، جداول، روابط
- **نسخ الردود** — زر نسخ فوري لكل رسالة
- **اختصارات لوحة المفاتيح** — Enter, Shift+Enter, Ctrl+Shift+N, /, Escape
- **بطاقات اقتراحات** — نقاط بداية سريعة للمحادثات الجديدة
- **SEO محسّن** — عناوين، وصف، وبنية دلالية
- **Firebase Analytics** — تتبع زيارات، رسائل، أجهزة، لغات

### 🔄 تغيّر (Changed)
- الانتقال من OpenRouter إلى **Groq API** لسرعة أعلى
- ترقية النموذج إلى **Llama 3.3 70B Versatile**
- إعادة تصميم الواجهة بالكامل — مستوحاة من ChatGPT/Claude
- تحسين System Prompt لردود أكثر دقة واختصاراً
