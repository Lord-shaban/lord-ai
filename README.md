## ✦ التقنيات

```
HTML5          ─  هيكل صفحة دلالي
CSS3           ─  متغيرات CSS، أنيميشن، تصميم متجاوب
JavaScript     ─  Vanilla JS بدون أي مكتبات
Firebase       ─  Firestore لتحليلات المستخدمين

Cloudflare     ─  استضافة سريعة عالمية
```
